    The Google Pixelbook Go is Google’s mobility-focused Chromebook designed with on-the-go users in mind. With Google Assistant built-in, it can help answer your questions, get things done, manage your schedule, and more. It’s accessible via its dedicated hotkey or by voice activation using the phrase, “Ok Google”. Specs-wise, it’s equipped with an 8th generation Intel Core i5 processor, 16GB of RAM, and a 128GB SSD. Should you need more storage space, you’ll be able to connect external storage solutions using its two USB Type-C ports. Its 13.3″ display, driven by integrated Intel UHD graphics, has a 1920 x 1080 Full HD resolution and supports multi-touch inputs, so you can easily write, draw, and navigate intuitively. Other integrated features include Wi-Fi 5 (802.11ac), Bluetooth 4.2, a backlit keyboard, a 1MP webcam, microphones, speakers, and a mic/headphone combo jack. The operating system installed is Chrome OS. Chrome OS : Using Linux as a base, Google designed Chrome OS to work exclusively with web applications. The user interface takes a minimalist approach, which greatly resembles the Google Chrome web browser. Chrome OS was built for people who spend a majority, if not all, of their time on the computer connected to the Internet. The only application that comes installed on Chrome OS is a special Google Chrome web browser with an integrated media player and file manager. All other applications are accessed by adding apps from the Google Chrome Web Store. Since Chrome OS relies entirely on Google’s web apps and access to Google’s services, it is highly recommended that you connect to the Internet to optimize your Chrome OS experience
