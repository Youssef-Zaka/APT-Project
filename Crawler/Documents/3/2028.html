COUNTER provides the standard that enables the knowledge community to count the use of electronic resources.

Known as the Code of Practice, the standard ensures vendors and publishers can provide their library customers with consistent, credible and comparable usage data.

 
 

For member libraries

COUNTER ensures libraries have reliable usage data for journals, e-books, databases and multimedia resources.

 Libraries

For member publishers & vendors

For publishers and vendors, COUNTER provides a Code of Practice, to ensure that their library customers receive reliable and consistent usage data.
