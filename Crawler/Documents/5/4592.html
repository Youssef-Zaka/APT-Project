Wikidata:Requests for permissions/Administrator
To request adminship for yourself or another user, please input the relevant name into the box below. If there has been a past request, successful or unsuccessful, please follow add a number − "Example 2" for a second request, "Example 3" for a third, and so on. Be sure to transclude the request on this page.


Old requests go to the archive. If you want to request administrator help, please post at Administrators' noticeboard.
Requirements for adminship

To be promoted, the following requirements must be met:

  • Nominations remain open for at least a full week.
  • To pass, the user needs:
    • at least 8 support votes
    • at least 75% of support or higher in pro/con voting
  • Only registered users with minimum 100 local non-automated edits can vote. The edits must be acquired before the voting started. Unregistered users can leave comments in the comments section.

Pi admin bot[edit]

Vote

RfP scheduled to end after 20 April 2022 17:58 (UTC)

  • Pi admin bot (talk • contribs • new items • new lexemes • SUL • Block log • User rights log • User rights • xtools)

I've been operating Pi bot since 2017, and it has made nearly 7 million edits on Wikidata so far. The latest task I've coded up, Wikidata:Requests for permissions/Bot/Pi bot 25, is designed to maintain Wikidata:Properties for deletion - and to simultaneously update Template:Watchlist summary/PFD. Which was all looking good and working fine for semi-automated test edits under my (admin) account, until I realised the template is admin-protected. Asking how to handle this at the administrators noticeboard favoured setting up an admin bot (over reducing protection of the template) - hence this request. The admin account needs to be separate from Pi bot's normal account for safety reasons, and it will be run in a separate directory on the Raspberry Pi. My main concern with proposing this admin bot is that just updating an admin-protected page might not be sufficient justification for having an admin bot account, but I expect it will only be a matter of time before I'll request to add more admin bot tasks to this account. For additional background, please see my user page, and my RfP. Thanks. --Mike Peel (talk) 17:58, 13 April 2022 (UTC)[reply]
