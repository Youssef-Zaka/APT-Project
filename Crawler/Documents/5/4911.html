This page is presently inactive and retained primarily for historical interest.
Extended uploader has been discontinued per community consensus and all user rights have been granted to both the autopatroller and the patroller groups.

Extended uploaders was a special user group for registered Commons users who are trusted to perform restricted file uploads. Extended uploaders are supposed to be familiar with the project scope, and especially with the file license policy.

Extended uploaders have the following user rights:

  • Upload files (upload)
  • Overwrite existing files (reupload)
  • Not be affected by IP-based rate limits (autoconfirmed)
  • Upload files from a URL (upload_by_url)
  • Uploading files from flickr using UploadWizard
  • Exempted from the abuse filter that prevents most users from uploading MP3 files on Commons.

If you would like to be added to the user group, please make a request at Commons:Requests for rights. Administrators and license reviewers can perform restricted uploads automatically.
