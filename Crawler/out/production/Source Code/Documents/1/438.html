Secure

Proactive defense against threats and careless users

Chrome has built-in and added protections to safeguard your organization from external and internal threats.

Sandboxing

Prevents malware and isolates malicious web pages that try to infect devices or steal data.

Regular updates

Protects users from the latest security vulnerabilities — no patching required.

Site isolation

Goes beyond sandboxing to further protect malicious code from spreading between processes on the same tab.

Password alert

Detects and alerts users when corporate password reuse is detected to protect enterprise accounts.

Extension security

Chrome allows users to benefit from extensions, while allowing admins to control extensions based on permissions and sites.

Management

Easy, centralized management

Streamlined management tools give admins full visibility and effective cross-platform controls.

Chrome Browser Cloud Management

Single destination for managing browser policies, settings and extensions across Chrome browser users.

Microsoft Active Directory support

For enterprises that prefer to use on-premise solutions, Chrome also supports Group Policy with hundreds of policies available for Windows.

Legacy Browser Support

Dynamically redirects to a legacy browser for specific apps as necessary, keeping users in a secure modern browser unless there is an exception.
