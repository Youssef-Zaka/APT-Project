About edit requests

What an edit request IS NOT for:
making test edits: use the sandbox
asking for help: go to the Help Desk
making a comment or starting a discussion: go to the talk page
requesting unprotection: go to requests for page protection

What an edit request IS for:
requesting straight-forward changes like grammar, spelling, formatting
requesting more substantive changes, if they are justified with reference to reliable sources, and not the subject of ongoing discussion

When submitting your request

  1. Please provide a specific description of the edit request
  2. Please don't copy the entire article into the request. Only copy the part you're changing.
  3. Please provide reliable sources to support your edit, if appropriate.
If you have a user account, log in first. If you do not yet have an account, you may create one (this is free and requires no personal information); after a while and some edits, you will be able to edit semi-protected pages. You might also be interested in the Wikipedia:Tutorial.
This is a talk page. Please respect the talk page guidelines, and remember to sign your posts by typing four tildes: ~~~~.
