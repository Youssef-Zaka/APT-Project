Pronunciation[change]

  • (US) enPR: äbʹsəlēt, IPA (key): /ˈɑbsəliːt/, SAMPA: /'Abs@li:t/
  • (UK) IPA (key): /ˈɒbsəliːt/, SAMPA: /'Qbs@li:t/
  • Audio (US)(file)

Adjective[change]

Positive
obsolete

Comparative
more obsolete

Superlative
most obsolete

  1. If something is not used anymore, or if something newer replaced it, it is obsolete.
