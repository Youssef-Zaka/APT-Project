Testbank contains randomized versions of this quiz for classroom use. For more information on printing these quizzes, see How to use testbank.

For on-screen use, this version of the quiz can be used as a pre-reading activity for Wikipedia:Computer. An excellent homework assignment might be to ask students to submit more quiz questions to Wikiversity.[1]
