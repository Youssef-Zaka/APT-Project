FAST API

Developer access to OCLC's Faceted Application of Subject Terminologies (FAST) subject headings data.

  • Status: Experimental
  • Sandbox access: No

What you get

  • Get access to the FAST heading set as Linked Data
  • Search FAST headings via assignFAST
  • Search FAST headings by geographic coordinates

Who can use it

The FAST API is available to anyone for non-commercial use.
