Usage

This documentation is incomplete.

This template allows displaying all or parts of the staff list, in a variety of formats. The format is set using the type= parameter.

  • type=showall displays a gallery of the entire staff, broken up into departments using headers.
  • type=departments_navbox generates a navbox of all departments.
  • type=department_gallery displays a gallery of the a single department, specified with section=. The size of the images in the gallery can be set by the first parameter.
  • type=teams_navbox generates a navbox for a single department, specified with section=.
  • type=team_list generates a simple list of all employees in a single department or team, specified with section=.

This template uses Module:Staff list. Information on the data format used in this template can be found at the documentation there.
