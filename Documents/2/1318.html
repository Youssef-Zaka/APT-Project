External links search

Jump to navigation Jump to search

A wildcard may be used at the start of the name only, for example, "*.wikimedia.org". To search for external links to pages on Meta-Wiki, start with

meta.wikimedia.org/

The search pattern is case-sensitive after the first slash (/) after the host- or domain-name. The “http://” protocol specification is optional; any other protocol (“https://”, etc.) must be given explicitly. For performance reasons, a limitation to specific namespaces is no longer possible. This functionality is, however, provided by toolforge:linksearch.

For more details, see Help:Linksearch and mediawiki:Help:Linksearch.
