British Library archived project page
This page has been created to act as a focal point for contributors improving articles related to the British Library. From May 2012 to April 2013, the BL hosted a Wikipedian in Residence, Andrew Gray (talk · contribs).

The current page (under construction) can be found here.

This task force is believed to be semi-active.
Although activity is slower than it once was, anyone is welcome to participate in the project.
Remove the |Semi-active parameter from this template if activity resumes or if this tag was changed in error. If almost no activity occurs in this task force, consider replacing this tag with {{WikiProject status|Inactive}}.

Activities[edit]

  • 8 October 2016 Europeana Sounds edit-a-thon at the British Library
  • AHRC podcasts on the Wikipedian in Residence project and how to use and contribute to Wikipedia
  • Wikipedia:Participation by academic projects - guidance on how research projects can engage with & use Wikipedia
  • request some digitised books! - now superseded by the Mechanical Curator collection on Commons.
  • Articles relating to the British Library
  • Training workshops - now completed
  • Events - now completed
  • One on one collaborations - currently inactive, but get in touch!
