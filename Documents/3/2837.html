Visual presentation of the initiatives.

This is a simplified version of the list of initiatives (those being: the key outcomes, changes or actions) embedded in the Wikimedia 2030 Movement Strategy recommendations. Initiatives from this list are part of the Movement Strategy implementation.

Priority initiatives[edit]

Initiatives prioritized for global coordination.
Main articles: Table of prioritization events and November global conversation report

Some of the initiatives in this list are referred to as "priority initiatives for global coordination". This means that those initiatives, during previous discussions, were identified as priorities that need to be implemented in coordination across the Wikimedia Movement. An example of these would be the Movement Charter, which obviously requires an implementation through a wide consultation with the different Wikimedia communities and stakeholders.

This does not mean, however, that only "prioritized initiatives" can be implemented. Communities and affiliates are at liberty to locally implement any of the initiatives listed below, and many of them have already identified their own priority initiatives that are not necessarily the same as the priorities for global coordination. People working on an initiative in their local community or region are encouraged to add details about their work in the initiatives pages that are listed below. If any given initiative is still missing a page, anyone is welcome to create it.
