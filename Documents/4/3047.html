Noun[change]

Singular
algorithm

Plural
algorithms

An algorithm
  1. (mathematics)(technical) An algorithm is a set of steps that reliably solve certain types of problems, such as sorting things alphabetically or doing long division.
    The new search algorithm was able to find a solution 5% faster than the previous algorithm.
