Wikidata:ORES

From Wikidata
Jump to navigation Jump to search

The Objective Revision Evaluation Service (ORES) is a service that provides AI scoring for revisions. For example, the score indicates the probability of an edit being vandalism. If you want to see what the API does, you can try scoring an edit like this:

https://ores.wikimedia.org/v3/scores/wikidatawiki/247205553

You can read more about ORES in Wikidata in this blog post (English).

Usage[edit]

ORES predictions are now integrated with new RecentChanges and Watchlist filters.

Contribute[edit]

There is some vandalism we miss and some non-vandalism that we flag. We'll be working on the prediction strategy to improve accuracy. In order to help us you can report mistakes flagged by ORES on this page (old reports). Also you can share common patterns in vandalism in Wikidata in this page.
