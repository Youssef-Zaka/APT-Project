 WikiProject India COVID-19 task force

WikiProject India COVID-19 task force is an initiative to create and enrich up-to-date database related to COVID-19 pandemic in India (Q84055514). If you are a Wikidata editor, please consider contributing, or encourage other editors to join. Have a look at the tasks below. Questions, comments, and discussion are welcome at the talk page.

Tasks[edit]

  • Update state and district level data everyday (co-ordinate with Bodhisattwa)
    • Add updated number of cases (P1603), number of recoveries (P8010), number of deaths (P1120), number of medical tests (P8011) for each items.
  • Create and give ideas about relevant SPARQL queries. (ping Mahir256)
  • Create SVG maps and upload them to Commons. (co-ordinate with KCVelaga)
  • Create items for daily Bulletins published by all state governments (co-ordinate with Gnoeee and Adithyak1997)
  • Add references about the cases.
    • Preferred references are official bulletins and reports published by Indian and State / UT's and almost all those are listed under Resource section. This can be added as reference with qualifier - stated in (P248)
    • Some State / UT's doesn't have official bulletins and reports published but having official dashboard. This can also be added as reference with qualifiers - reference URL (P854) and retrieved (P813)
    • If both bulletins and dashboard are not available add references from reputed sources. For knowing how to add external sources check here.
  • Create statewise delta column to track state wise progression (by Namlasdaw)
  • And finally update the Wikidata items regularly and update the details in Google sheet
  • Build a dashboard - development phase -Country wise and State Wise dashboards and -Interactive Indian Map

Add Wikidata values to Wikipedia Infobox[edit]

Wikidata values can be used inside an infobox. So that any changes that has been made in Wikidata will reflect in the Wikipedia article.

Steps to add Wikidata values in Wikipedia Infobox
  1. In regional language Wikipedia's some Templates and Modules must be imported first (if not available).
  2. Import Template:Stringsplit, Module:String2 along with its document and Sandbox pages.
  3. Import / Update Module:WikidataIB
  4. After succesfully importing these go to the article page in the regional Wikipedia;
    • Change the value of confirmed_cases in the Infobox pandemic to {{wdib|fwd=ALL|P1603|qual=P585|list=p-1|df=dmy}}
    • Change the value of recovery_cases to {{wdib|fwd=ALL|P8010|qual=P585|list=p-1|df=dmy}}
    • Change the value of deaths to {{wdib|fwd=ALL|P1120|qual=P585|list=p-1|df=dmy}}
    • Change the value of active cases to {{#expr: {{formatnum:{{wdib|fwd=ALL|noicon=true|P1603|list=p-1}} - {{wdib|fwd=ALL|noicon=true|P8010|list=p-1}} - {{wdib|fwd=ALL|noicon=true|P1120|list=p-1}} |R}} }}
  5. Preview the edit and Publish if there is no errors due to the module and Template.

Thanks to User:RexxS who helped in creating the code. If anyone has any problems with step 3 (or any other part), please feel free to ping User:RexxS who will be happy to help resolve any issues.
