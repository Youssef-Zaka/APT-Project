이 용어 사전은 사용자가 재단의 개인정보처리방침을 볼 때 잘 모르는 일부 용어를 이해하기 쉽게 설명하지만 공식 개인정보처리방침에는 포함되지 않습니다.

API

"응용프로그램 프로그래밍 인터페이스", 또는 "API"는 소프트웨어 요소 간 교신을 위한 교신 프로토콜입니다. 예를 들어 미디어위키 API는 제삼자 소프트웨어 도구가 위키미디어 사이트들을 포함한 미디어위키 사이트와 교신하여 자동적이며 원격적으로 문서를 읽거나 고치는 등의 특정한 행동을 할 수 있게 합니다. 더 알아 보기.

브라우저

"브라우저"는 일반적으로 인터넷에 접속하기 위해 사용하는 소프트웨어 응용 프로그램입니다. 브라우저의 주요 예시로는 크롬, 파이어폭스, 사파리, 인터넷 익스플로러 등이 있습니다. 더 알아보기.

쿠키

"쿠키"는 사용자가 위키미디어 사이트들로 접속할 때 재단이 일반적으로 사용자 인증, 또는 간혹 추적을 위해 컴퓨터, 휴대전화, 기타 기기로 전송하는 작은 데이터 파일입니다. 모든 쿠키는 특정 기간 이후에 만료되지만, 그 기간은 쿠키가 무엇을 위해서 사용되며 브라우저가 어떻게 설정되어 있는지에 따라 다릅니다. "세션" 쿠키는 일반적으로 사용자가 웹 브라우저나 모바일 응용프로그램을 종료할 때 만료됩니다. "지속성" 쿠키는 사용자가 브라우저나 모바일 응용프로그램을 닫은 이후에도 자신의 기기에 남아 있습니다. 지속성 쿠키는 재단이 설정한 기간이 지나(거나 사용자가 수동으로 쿠키를 삭제할) 때 만료됩니다. 더 알아 보기.

사용자는 브라우저 설정에서 쿠키를 지우거나 비활성화할 수 있습니다.

사용자 추적방지

"사용자 추적 방지(Do not track)"나 "DNT"는 사용자의 웹브라우저에서 방문하고 있는 웹사이트로 분석 서비스나 광고 네트워크, 소셜 플랫폼과 같은 자신이 방문하지 않는 제삼자 웹사이트가 자신을 추적하는 것을 원하지 않음을 밝히는 방식입니다. 이 메커니즘을 사용하면 사용자의 브라우저에서 신호를 보내서 자신의 개인정보, 특히 자신의 온라인 활동과 네트워크 상호작용이 제삼자에게 전달되지 말아야 한다는 사용자의 소망을 표현합니다. 브라우저에서 발신된 사용자 추적 방지 신호를 받게되면 해당 웹사이트를 소유한 개인이나 단체는 위에서 언급된 요청을 받아들이거나 무시할 수 있습니다.

GPS

“위성 위치 확인 시스템”인 “GPS”는 우주 기반 위성 항법 시스템으로 차량 항법 시스템이나 휴대 전화와 같이 신호를 수신하는 장치의 위치와 시간 정보를 제공합니다. 더 알아보기.

인터넷 프로토콜 주소

인터넷 프로토콜 주소("IP 주소")는 인터넷에 연결된 특정 기기에 부여된 유일한 번호입니다. 자신의 IP 주소는 프록시 서버나 회사 네트웨크와 같은 많은 변수에 따라 특정한 기기에서 항상 동일하지 않을 수 있습니다. 그러나 보통 IP 주소는 자신의 기기와 자신의 인터넷 서비스 제공자가 있는 지리적 위치에 따라 할당되며, 대략적인 자신의 실제 위치를 표현할 수 있습니다. 더 알아보기.

자바스크립트

"자바스크립트"는 많은 웹사이트에서 사용되는 표준 소프트웨어 프로그래밍 언어입니다. 더 알아보기.

로컬 저장소

“로컬 저장소”는 웹사이트에서 정보를 "로컬 영역에"(웹사이트의 서버가 아닌 사용자 기기에) 수집하고 저장한 후 나중에 그 정보를 다시 읽을 수 있도록 하는 방식입니다. 예를 들어, 로컬 저장소를 사용하면 사용자의 방문 기록이 자신의 컴퓨터에 저장되고 기록된 이후 재단으로 제공될 수 있습니다. 이것은 재단이 주요 사용 통계(방문 횟수) 정보를 받을 수 있게 하지만, 각각의 방문이 언제 이루어졌는 지에 대한 특정 정보는 결코 재단으로 전송되지 않습니다.

메타데이터

“메타데이터”는 (사진이나 비디오 같은) 특정한 파일에 대한 추가적인 정보로, 보통 사진을 찍은 기기의 제조사와 모델, 사진이 찍힌 날짜와 시간, 노출 시간, 렌즈 초점 거리, ISO 속도 규격, F-값 등을 포함합니다. 일부 메타테이터는 기기에서 자동으로 포함하며 일부는 기기 사용자가 기록합니다. 더 알아보기.

운영 체제

“운영 체제”는 사용자의 하드웨어 자원을 관리하고 파일 관리, 키보드 입력 인식, 모니터로 출력 보내기와 같은 기본 작업을 수행하는 소프트웨어입니다. 운영 체제에는 리눅스, iOS, 윈도, 맥 OS X, 안드로이드 등이 있습니다. 더 알아보기.

프록시 서버

“프록시 서버”는 사용자 기기와 사용자가 정보를 요청(웹페이지, 파일 등의 연결)하는 서버 사이에서 중개하는 역할을 하는 서버입니다. 더 알아보기.

추적 픽셀

“추적 픽셀”(또는 “웹 신호”, “투명 GIF”, “맑은 GIF”, “픽셀 gif”, “pixel 태그”라고도 함)은 재단에서 위키미디어 사이트의 활동이나 재단에서 보내는 알림에 기초한 활동을 추적할 수 있도록 하는 작고 보이지 않는 이미지입니다. 추적 픽셀이 일반적으로 광고와 관련되어 있지만, 재단은 추적픽세를 광고를 위해 사용하거나 추적 픽셀을 통해 수집된 정보를 판매하거나 대여하지 않습니다. 추적 픽셀을 통해 수집된 정보는 통합된 형식으로 개인정보처리방침에 따라서라야만 제삼자와 공유할 수 있습니다. 추적 픽셀은 재단이 특정 기능, 알림, 제품이 효과적인지와 그것이 개선될 수 있을지를 이해하는 데에 도움이 됩니다. 위키백과에서 더 알아보기.
