License template for images uploaded from Unsplash.com. This template automatically includes {{Cc-zero}}.

This template should only be used for files which were published on Unsplash.com prior to 5 June 2017. Files published after 4 June 2017 are not available under CC-Zero and do not comply with Commons:Licensing. See this August 2017 discussion on Commons:Village pump.

The template automatically sets the following categories: Category:Images from Unsplash

Usage

{{Unsplash}}

Note for license reviewers: use caution when reviewing files. On an individual photo page on Unsplash.com, confirm that the publication date is 4 June 2017 or earlier (viewable from the (i) information symbol in the bottom right-hand corner). Photos published on 5 June 2017 or after do not pass license review and should be nominated for deletion instead.

The template takes no parameters.

Additional information

The template is intended to be used in the following namespaces: the File namespace

Localization

Deutsch ∙ English ∙ español ∙ македонски ∙ 中文（简体） ∙ 中文（繁體） ∙ العربية ∙ 中文 ∙ italiano ∙

This template is localized through {{Autotranslate}}. The layout of the template can be found under Template:Unsplash/layout.

To add your language to the list of languages supported by this template, please copy the code of Template:Unsplash/en (or any other language version you prefer to translate) and replace the text strings in there (the form below can be used to create a translation, the English version is preloaded in the edit box). Please change the parameter lang from en (or whatever language you are translating) to the language code of your language.
