This is an automatically updating directory of WikiProjects, including metrics on each project's activity, lists of active participants, and related WikiProjects. Find a project using the links below!

To include a WikiProject in the directory, place the main project page in a subcategory of Category:WikiProjects by topic. Note that the bot that updates this directory is currently not maintained; information may be inaccurate or not present.

A user is counted as a participant in a WikiProject if they have made at least two edits to the WikiProject's page, its talk page, or any subpages thereof within a rolling 90-day period.
