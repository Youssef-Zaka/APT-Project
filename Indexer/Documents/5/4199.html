Incorporating Visual Layout Structures for Scientific Text Classification

@article{Shen2021IncorporatingVL,

Classifying the core textual components of a scientific paper—title, author, body text, etc.— is a critical first step in automated scientific document understanding. Previous work has shown how using elementary layout information, i.e., each token’s 2D position on the page, leads to more accurate classification.
