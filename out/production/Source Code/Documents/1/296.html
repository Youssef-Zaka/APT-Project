This is a draft list of pages at which individual projects should be notified of foundation announcements. Projects which use different venues for different kinds of announcements should choose a page where they will be notified and handle any further copying and moving themselves. This list has been used to create this list of targets for automatic global message delivery.

Note: If there is no local page selected for announcements, the MediaWiki default or English-language version will be used.

See also outreach:Best practices for reaching out to projects in multiple languages (should probably be merged to a Meta page and updated).
