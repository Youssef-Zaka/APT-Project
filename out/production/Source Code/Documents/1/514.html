Mac VS Windows

July 5, 2016July 5, 2016 Matthew 0 Comment


The battle of Mac vs Windows

 This as old as computers themselves. Each product has its faithful users that would rather debate then admit the other product might be better than theirs in some way. So where does that leave the average computer user? Going through articles and discussions, both sides are subtle on proving their product is better. It’s almost impossible to find a neutral opinion on the topic. It’s true that both operating systems have their ups and downs, and we could have a long debate on what the differences are.

To a person just starting to learn a computer for the first time, discussing software, hardware, customization, and design can be very over whelming. Yes, all those things are important. But really how important? If you’re just learning to use computers for the very first time all the technical talk can be exhausting. Yes, that’s stuff you need to learn, but not now, no matter what computer you decide on. But for now you need to think about what will fit your best needs, and what’s easiest for you to learn to operate.

When people ask me for advice as to which type of computer to buy, a Windows or a Mac. I will ask questions and ask the user what they will use it for. More than likely, a person at the start of their computer experiance just want to accomplish a the basic tasks; check email, get on the internet, check Twitter and Facebook, play games, and maybe do some other type of basic task. Today we will help you to try to come to a descion with what computer system is easiest to learn and use. Even this article to help you can lead to a lot of discussion, but it’s always good to have somewhere to start.

Both Windows’ and Mac systems use a Graphical User interface (GUI). This means you have windows, icons, and other easy graphics that allow you operate programs and files through a visual que. But, the systems are very different in the way they operate.

The desktop of a windows 7 computer.

 

Desktop of Mac computer with Leopard OS

Macs use very easy to lean operating system, that does not require alot of technical knowledge, making them easier to understand. The dock at the bottom of a Mac, seen at the bottom of the screen in the photo above, uses big and very colorful icons, and makes programs that are used frequently very easy to use. Along wit this the Mac has a mouse that has a one button click. Removing programs and files on a Mac is very straightforward. Just drag and drop. When it comes to windows it would require an installer or uninstaller, and a lot of times these don’t necessarily remove all traces of the file or software the way they should do. 

The windows on a Mac and Windows computer allows you to see every open window on your computer, create as many desktops as you want, and organize your work so that everything you need is accessible. Mac computers also make it very easy to apply a change, where most of the time you just make the change and exit the window and your good to go. The Mac operating system seems to be more responsive and requires less input from the users. A very good feature with a Mac is Apple makes its own hardware, software, and accessories, and gives the consumer a one stop shopping place. Now for a new user this is a very good thing but for an advanced user they like more options. For a Mac user, it’s good because you don’t need to make decisions about what software and extra components to get and then deal with the challenge of installing it later. Mac users simply power up the computer and use them straight out the box. If an Apple user has a problem with a part, there a single point of contact: Apple. Windows-based have hardware and software made by multiple outside vendors. Apple also has their customer support center based all in the United States, unlike a lot of other manufacturers who outsource support to other countries.

Both operating systems take time to learn, and different people will always prefer one system over another for various reasons. Since the 2 operating systems varies so much, A user may ave a hard time using the other operating system. Because of this, if you have any type of previous experience with either operating system, you should consider staying to the system you know about. If you have a Notebook, Tablet or smart phone running windows, then just buy a Windows-based operating system. If you have an iPad or iPhone, then buy a Mac based operating system. Sticking with what you know will help you be a lot more comfortable. I’ve had the pleasure to work and learn on both systems since I was a young child. So, for a first time user I do recommend a Mac computer.
