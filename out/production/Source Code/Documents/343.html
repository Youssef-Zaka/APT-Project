維基媒體項目

維基媒體基金會支持全世界數十萬人創建歷史上最大的免費知識項目。 志願者的工作幫助全球數百萬人發現信息，貢獻知識，並與他人分享。

參考

  • 維基百科

    全世界的知識

    維基百科是由世界各地的志願者用300種語言編寫的免費協作百科全書。

  • 維基教科書

    電子書教科書和註釋文本

    維基教科書旨在建立一系列免費電子書資源，包括教科書，註釋文本，教學指南和手冊。

  • 維基詞典

    超過170種語言的字典

    維基詞典是一本免費的多語言詞典。 該項目旨在描述所有語言的所有單詞。 它包括語言資源，如詞庫，押韻指南和語言統計。

  • 維基語錄

    查找您喜愛的書籍，電影，作者等的語錄

    維基語錄是一個收集超過75種語言來自著名人物和創意作品的引文庫。 它包括諺語，助記符號和口號。

系列

  • 維基共享資源

    6800萬張圖像，照片，視頻和音樂文件，並不斷發展壯大

    維基共享資源是世界上最大的免費圖庫，照片，圖畫，視頻和音樂庫。

  • 維基文庫

    免費圖書館

    維基文庫是一個自由許可的源文本和歷史文檔庫，包括詩歌，政府文檔，許多國家的憲法和一般文獻。

  • 維基學院

    獲取任何學習水平的學習資源，項目和研究

    維基學院致力於學習資源，學習項目和研究，以用於所有級別，類型和教育方式。

  • 維基物種

    物種目錄

    維基物種是分類學物種數據庫，涵蓋動物界，植物界，真菌，細菌，古細菌，原生生物和所有其他形式的生命的生活和化石形態。

技術

  • 維基數據

    協作編輯的結構化信息數據庫

    維基數據充當維基媒體項目結構化數據的中央存儲。將數據結構化為機器可讀格式，可以更輕鬆地查看，搜索，編輯，策劃，使用和重用文件。

  • MediaWiki

    Wiki軟件開發

    MediaWiki是免費的開源wiki軟件，任何人都可以使用和開發。 它是構建維基媒體項目的平台。

指南

  • 維基導遊

    終極旅遊指南

    維基導遊旨在創建世界上最大的免費，完整和最新的全球旅行指南。

  • 維基新聞

    自由的新聞來源

    維基新聞提供商業新聞網站的免費內容替代品。

合作

  • 元維基

    用於全球協作的項目協調軟件工具

    元維基是一個項目，用作各種協調和組織任務的中心樞紐，例如影響多個wiki或計劃即將發生的事件的討論。

幫助我們釋放世界的知識。
