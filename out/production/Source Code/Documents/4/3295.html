    Semantoya (talkcontribs)

    Hi,

    From a user perspective several aspects can be improved in Wikimedia.

    Firstly, most of the content is still plain text, lacking any structure to be machine interpretable. To improve this, we really need a controlled vocabulary, that is, term definitions from a central repository. Incorporation of Wikibase items would be a nice approach. However, this is still in an immature stage. Other options of the mediawiki-family are unsuited because these are plain text solutions and therefore not machine interpretable.

    Secondly, the possibility to deliberate on the content of a page shall be improved. Yes, there is a discussion page but to my knowledge it is not possible to point a bookmarked sections and paragraphs a page.

    Frans
