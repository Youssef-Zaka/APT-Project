Featured Webinar

Snowpark for Python: Using Anaconda in Snowflake

Teams working with massive amounts of data are rapidly adopting Python to accelerate data pipeline and machine learning workflow development. Yet teams are often bound by the operational overhead required to manage infrastructure and environments. Pairing Snowflake’s near-unlimited access to data and elastic processing engine with Anaconda’s repository and package manager can change that – so more time can be spent on building. Join us to learn how Snowflake makes the most popular packages from Anaconda seamlessly accessible for processing data at any scale.
