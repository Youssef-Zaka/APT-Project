Inner loop container developer productivity with Google Cloud

This session addresses a day in the life of a container developer and the inner loop's best practices to significantly improve developer productivity for building, testing, and debugging containerized applications. Container development is slow in the inner loop as the code takes a few minutes to be built into a container. Consequently, developers may not use containers for application development. This causes disparity between development and the higher environments where the code actually runs. We offer a solution to address these challenges that impacts almost all developers who are adopting containers. The solution involves products such as Cloud Code, Artifact Registry, and GKE to speed up your application development. In this session, Veer Muchandi, Solution Manager, Google Cloud will show you a quick demo of how application development can be much easier and faster with this solution.
